# For-the-Kim
---
## 작업 의도
[생활코딩 WEB1](https://opentutorials.org/course/3084) 수업을 듣고 실제로 내 일상에 적용을 해보기 위해

## 작업 내용
군대에 가 있는 지인을 우상화, 신격화하는 웹사이트를 제작

## 사용된 도구들
HTML, CSS를 사용 중이다.

## 개선해야 할 점
- 게시판, 글쓰기 시스템 구현
- 전역 일자에 맞춰 카운트 다운
- 배웠던 미디어 쿼리를 이용한 반응형 디자인 구현
- 전체적인 디자인을 깔끔하게?
- 광고 달기?
